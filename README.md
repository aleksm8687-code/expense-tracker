# Менеджер личных расходов

## Описание

«Менеджер личных расходов» — это консольное приложение на Python для учета ежедневных трат.
Проект создан как учебная работа для освоения командной разработки с Git и GitHub.

Приложение умеет:

- добавлять расход;
- показывать список всех расходов;
- считать общую сумму расходов;
- показывать статистику по категориям;
- удалять расход по номеру;
- хранить данные в JSON-файле.

## Меню приложения

```text
1. Добавить расход
2. Показать все расходы
3. Показать общую сумму расходов
4. Статистика по категориям
5. Удалить расход
6. Выход
```

## Формат хранения данных

Данные сохраняются в файле `expenses.json`.

Пример записи:

```json
{
  "category": "Еда",
  "description": "Обед в кафе",
  "amount": 2500,
  "date": "2026-06-01"
}
```

## Запуск проекта

```bash
python main.py
```

## Запуск тестов

```bash
python -m pytest
```

## Git-ветки проекта

В проекте использованы ветки:

- `feature/add-expense` — добавление расходов;
- `feature/list-and-stats` — вывод списка, общей суммы и статистики;
- `feature/delete-expense` — удаление расходов;
- `feature/tests` — модульные тесты;
- `feature/date-validation` — ветка для Pull Request и закрытия Issue.
